## 4. Tài liệu học tập theo cấp độ

### Beginner (Người mới bắt đầu)

- **SAP Learning**: Tìm learning journey cho module muốn học → làm bài tập thực hành trong SAP Learning System
- **openSAP**: Khóa học "SAP S/4HANA Overview" (miễn phí, 3 tuần)
- **SAP Help Portal**: Đọc "Getting Started" guide cho từng module
- **SAP Community**: Theo dõi tag tên module (VD: #S4HANACloud #MM)
- **SAP Best Practices Explorer**: Xem scope items và process flow

### Intermediate (Đã biết cơ bản)

- **SAP Learning**: Hoàn thành Associate learning journey → thi chứng chỉ
- **SAP Press**: Đọc sách chuyên sâu cho module (VD: "Sales and Distribution in SAP S/4HANA")
- **SAP Community**: Đọc blog, tham gia discussion, hỏi câu hỏi
- **SAP Fiori App Library**: Khám phá Fiori app cho module của mình

### Advanced (Chuyên sâu)

- **SAP Certification**: Thi Professional / Specialist certificate
- **SAP Press**: Sách nâng cao (VD: "ABAP Cloud", "Extensibility for SAP S/4HANA")
- **SAP Community**: Viết blog, trả lời câu hỏi
- **SAP API Business Hub**: Nghiên cứu released APIs cho module
- **SAP UCC / CAL**: Thực hành trên hệ thống thật

## 6. Mẹo học tập hiệu quả

### Nguyên tắc 80/20 cho SAP

80% nghiệp vụ hàng ngày chỉ dùng 20% kiến thức SAP. Tập trung vào 20% đó trước:

| Module | 20% quan trọng nhất (học trước) | 80% còn lại (học sau) |
|--------|-------------------------------|---------------------|
| **SD** | Sales Order, Pricing, Delivery, Billing | Credit memo, Rebate, Consignment |
| **FI** | GL, AP, AR, Bank, Business Partner | Asset, Closing, Consolidation |
| **MM** | PO, PR, GR, IR, Material Master | Stock Transport, Consignment |
| **CO** | Cost Center, Internal Order, Product Costing | CO-PA, Material Ledger |
| **PP** | Production Order, BOM, MRP, Work Center | REM, Kanban, Variant Config |
| **QM** | Inspection Lot, Inspection Plan, Usage Decision | Certificate, Audit |
| **PM** | Maintenance Order, Equipment, Notification | Predictive Maintenance, EAM |
| **WM** | Warehouse, Transfer Order, Picking | EWM, Yard Management |

### Phương pháp học

1. **Học theo quy trình (Process-based)**: Không học từng transaction riêng lẻ, mà học theo business process (VD: Order-to-Cash cho SD, Procure-to-Pay cho MM)
2. **Thực hành liền tay**: Đọc xong lý thuyết → làm ngay trên SAP Learning System
3. **Học từ lỗi**: Khi gặp lỗi SAP, tìm hiểu nguyên nhân → giải pháp → ghi chép lại
4. **Kết hợp nhiều nguồn**: SAP Learning + SAP Community + SAP Press = kiến thức toàn diện
5. **Dạy lại người khác**: Giải thích cho đồng nghiệp hoặc viết blog là cách học nhanh nhất

### Cross-module Integration Paths

Học SAP theo business processes cross-module thay vì từng module riêng lẻ:

| Integration Path | Modules liên quan | Business Process |
|-----------------|-------------------|------------------|
| **Order-to-Cash (OTC)** | SD → FI → CO | Từ đơn hàng → giao hàng → hóa đơn → thanh toán |
| **Procure-to-Pay (PTP)** | MM → FI → CO | Từ yêu cầu mua → PO → nhập kho → hóa đơn → thanh toán |
| **Plan-to-Produce (PTP)** | PP → MM → QM | Từ kế hoạch sản xuất → MRP → sản xuất → kiểm tra |
| **Maintain-to-Close** | PM → MM → FI → CO | Từ bảo trì → spare parts → chi phí → phân bổ |
| **Project-to-Report** | PS → SD → FI → CO | Từ dự án → billing → doanh thu → lợi nhuận |
| **Hire-to-Retire** | HCM → FI → CO | Từ tuyển dụng → lương → chi phí nhân sự |
| **Import-to-Export** | GTS → MM → SD → TM | Từ nhập khẩu → mua hàng → bán hàng → vận chuyển |
